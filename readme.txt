
# C-SlideShow version 2.0


## 基本操作

* マウスホイール・・・進む/戻る
* Shift + マウスホイール・・・画像一枚分だけ進む/戻る
* 右クリック + マウスホイール・・・拡大縮小


## 利用条件

以下に該当する方は利用禁止とさせていただきます。

1. PeerCastを捨てた配信者(抜け忍)
2. 不具合が起きた時に、作者に謝罪と賠償を求める人
